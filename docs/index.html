Welcome to the new Shatter website!

Shatter

Shatter is a way to edit segments for the 2014 mobile game Smash Hit using Blender 3D.

Download add-onManual

Create the levels of your dreams

You can create basically any segment you want, so long as it meets Smash Hit's limitations.

Import and analyse

You can import and analyse segments to learn how they are made and edit them.

Quickly test your changes

You can quickly test changes that you make to your segments using an easy to set up client and server.

Generate repetitive structures

Autogen, a new feature, allows you to automatically generate structures that are common in levels.

Light segments like never before

You can use advanced lighting to set the mood of your levels more percisely.

Ease of use is a priority

We are focused on creating an easy to use level editor and update Shatter to make it easier to use.

Tweak Tool

The Smash Hit Tweak Tool allows you to make patches to libsmashhit.so that change the gameplay.
